# toio PlayGround Card Counter (Unofficial) 
## トイオ・プレイグラウンド検品ツール (非公式)

Unofficial toio Playground Card Counter for inspecting card sets in workshops and classrooms.  
ワークショップや教室での枚数チェックに最適な、非公式のトイオ・プレイグラウンド用カード枚数検品ツール。

[**👉 アプリを試してみる**](https://akichika.github.io/toio-playground-card-counter-unofficial/)

※Web Bluetoothを使用しているため、Chrome/Edge等の対応ブラウザでアクセスしてください。SafariやiPhone/iPadのChromeは非対応です。
※iOS/iPadOSの場合は [Bluefy](https://apps.apple.com/jp/app/bluefy-web-ble-browser/id1492822055) 等のWebBluetooth対応ブラウザが必要です。


## 💡 これは何？
「トイオ・プレイグラウンド」シリーズに付属する大量のカード（ベーシック48枚、アドバンス96枚）を、Bluetooth接続した toio™ コア キューブでタッチするだけで効率よくカウント・検品できるWebアプリです。

## ✨ 主な機能
- **自動カウント**: カードを読み取ると即座にチェック。不足だけでなく複数セット混入による枚数超過も検出。
- **表示切り替え**: カードデザインを再現した「画像表示」と、全体を俯瞰できる「リスト表示」を切り替え可能。
- **データ管理**:
  - 検品結果をコメント付きで履歴保存（ブラウザのローカルストレージ活用）。
  - 検品結果をCSV/JSONでエクスポート・インポート。
- **マルチデバイス・OS対応**: Windows/Mac/Android (Chrome/Edge)。
  - ※iOS/iPadOSの場合は [Bluefy](https://apps.apple.com/jp/app/bluefy-web-ble-browser/id1492822055) 等のWebBluetooth対応ブラウザが必要です。
- **複数キューブ接続**: 最大8台まで同時接続可能。複数人での分担検品や、両手での高速スキャンに対応。
- **自走スキャン**: 並べたカードの上をキューブに自走（駆動）させながらの読み込みも可能。

## 🚀 使い方
1. [アプリ](https://akichika.github.io/toio-playground-card-counter-unofficial/)を開く。
2. **「toio キューブ 接続」**ボタンからキューブをペアリングします。
3. 手元のカードをキューブで次々にタッチしていきます。
4. 全てのカードが揃うと「全数OK」が表示されます。
